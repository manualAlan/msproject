This master’s project develops and validates a real-time RF waveform control approach for trapped-ion
quantum experiments using the ARTIQ control ecosystem and the Sinara Phaser module.
The motivation is that high-fidelity quantum operations require deterministic timing and repeatable phase
relationships across multiple RF and optical control pathways.

The work reported here includes: (i) a concise systems-level framing of trapped-ion quantum control
requirements and the role of RF-driven acousto-optic modulators (AOMs); (ii) a technical overview of ARTIQ’s
host/kernel execution model and RTIO timing semantics; (iii) a specification- and architecture-grounded
comparison between Phaser (AWG-class DAC + digital upconversion) and Urukul (DDS-class synthesizer);
(iv) an implementation layer that supports multitone output, precision phase control, and an experiment-facing
software interface; and (v) a measurement plan and validation templates for timing accuracy, phase stability,
and spectral purity.

Key contributions of this project are:
1) a multitone generation workflow for Phaser supporting several independently controlled tones with configurable frequency, amplitude, and phase,
2) a precision phase-control methodology compatible with ARTIQ RTIO determinism, including explicit phase reference times and controlled updates without unintended discontinuities, and
3) an experiment-facing interface that abstracts device configuration, waveform scheduling, and metadata logging for reproducible experiments.

These contributions establish a thesis-ready framework for presenting later quantitative timing, phase-stability, and spectral-purity results once the final measurement data are inserted.
